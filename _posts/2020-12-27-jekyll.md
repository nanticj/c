Jekyll is a static site generator.

# Installation
```bash
gem install bundler jekyll
```

# Commands
```bash
# Create a new project.
jekyll new MY-AWESOME-SITE

# Install a Gemfile.
bundle install

# Execute jekyll serve: http://localhost:4000
bundle exec jekyll serve
```
