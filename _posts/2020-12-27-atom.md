A hackable text editor for the 21st Century.

# Installation
## Windows (administrative powershell)
```powershell
choco install atom
```

# Settings
## Themes
```
File > Settings > Themes
  UI Theme: One Light
  Syntax Theme: Solarized Light
```

## Install
```
File > Settings > Install 
  Search 'script' and install
  Run(Shift + Ctrl + B)
```
